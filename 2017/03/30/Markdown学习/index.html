安装Evernote插件

把上面的插件安装好之后，其他的插件就都比较好安装了。

  • ctrl + shift + p 打开 Sublime Text 命令窗口
  • 输入install package，Enter进入插件安装界面
  • 输入evernote，Enter即可完成安装

到此处，Evernote插件就安装完成了，接下来就是绑定你的账号，获取印象笔记应用授权。

获取印象笔记应用授权的方法

国际版印象笔记客户打开
将 Evernote 网页中的 Developer Token 和 NoteStore URL 添加到 Sublime Evernote 插件的设置文件 Preferences > Package Settings > Evernote > Settings - User：

注意：

  • token 是以S=开头的一串字符串
  • noteStoreUrl 是一段 http 地址，你需要手动将https替换成http
  • 保存设置文件（可能还需要重启你的Sublime Text）之后，尝试打开一个笔记以确保你的印象笔记能正常工作

到此处印象笔记应用授权就结束了，接着就是验证一下就可以了。
验证方法：

  • ctrl + shift + p 打开 Sublime Text 命令窗口
  • 输入 Evernote，就会看见 Evernote 插件的许多命令
  • 点击 Evernote:list recent notes，如果看到罗列出最新的笔记，则说明授权成功。

    参考：sublime-evernote

好现在就已经可以使用印象笔记，但是为了有更好的体验，需要设置一些快捷键，因为默认的Evernote插件没有绑定任何快捷键，所有的操作都要通过ctrl + shift + p输入Evernote后选择相应功能进行。

经网上查询，这样的配置比较方便记录。设置方法如下：
Preferences > Key Bindings - User 关联快捷键

上面代码的意思是：

  • 先按 ctrl+e, 再按ctrl+s 后，会将笔记保存并且同步到印象笔记
  • 先按 ctrl+e, 再按ctrl+o 后，会打开笔记列表，然后就可以打开想要的笔记
  • 先按 ctrl+e, 再按ctrl+u 后，会更新笔记到原位置
